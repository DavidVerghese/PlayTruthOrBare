# SessionTogether

How to sign up 

1. In Postman, go to http://localhost:3000/api/sign-up
2. Select 'post request'. 
3. The message body of the request should be in this format: 
{
         "username": "username",
      "email": "email address",
      "password": "password"
  }
5.  You will receive a JSON webtoken
